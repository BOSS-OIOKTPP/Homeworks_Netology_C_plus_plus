# Дополнительные библиотеки

## Установка Boost

1. Скачайте Boost 1.90:  
	https://archives.boost.io/release/1.90.0/source/boost_1_90_0.zip

2. Распакуйте в папку, например:  
	C:\Libs\boost_1_90_0


# Настройки проекта в Visual Studio

1. Открыть свойства проекта 
    → C/C++ 
          → Общие:
	- в поле «Дополнительные каталоги включаемых файлов» написать:
	..\IniFiles;..\..\..\Libs\boost_1_90_0;

          → Язык:
	- в поле «Стандарт языка С++» выбрать  Стандарт ISO C++ 20 (/std:c++20)

          → Командная строка:
	-   в поле «Дополнительные параметры» написать     /utf-8

    → Компоновщик
          → Ввод:
	- в поле «Дополнительные зависимости» написать:
	$(CoreLibraryDependencies);%(AdditionalDependencies);ws2_32.lib;


 и нажать OK.
