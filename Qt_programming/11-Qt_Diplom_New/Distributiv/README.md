# Distributiv

Здесь лежит финальный комплект для проверки диплома.

## Что должно быть в этой папке

- `indexer.exe` — приложение индексатора.
- `search.exe` — приложение поисковика.
- `config.ini` — файл настроек (БД, каталоги, расширения).
- DLL-файлы Qt и драйвера БД, необходимые для запуска.
- `README.md` — инструкция по запуску и настройке.

## Рекомендация по оформлению

В `README.md` опишите:
- какие зависимости нужны (Qt, PostgreSQL);
- как настроить `config.ini`;
- порядок запуска: сначала `indexer.exe`, потом `search.exe`;
- пример тестового запроса и ожидаемый результат.
