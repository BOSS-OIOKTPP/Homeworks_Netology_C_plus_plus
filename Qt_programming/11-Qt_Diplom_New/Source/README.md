# Source

В этой папке лежат исходники по этапам диплома.

## Структура

- `1.DatabaseManager` — модуль работы с БД (Qt console).
- `2.Indexer` — консольный индексатор документов.
- `3.SearchGui` — графический поисковик на Qt.
- `4.Integration` — сборка частей в единый проект.

Правило: каждая часть должна быть самостоятельной и проверяемой отдельно.
