# Distributiv

В этой папке находится готовый комплект для запуска дипломного проекта.

## Назначение программ

- `indexer.exe` — индексатор. Обходит каталоги из `config.ini`, читает текстовые файлы и записывает в БД слова и их частоту.
- `search.exe` — поисковик с графическим интерфейсом. Выполняет поиск по БД и показывает документы, отсортированные по релевантности.

## Как запускать

1. Запустите `indexer.exe` и дождитесь завершения индексации.
2. Запустите `search.exe`, введите запрос и нажмите `Искать`.

## Настройка config.ini

Файл `config.ini` должен лежать рядом с `indexer.exe` и `search.exe`.

Структура:

```ini
[Database]
Host=localhost
Port=5432
Name=search_db
User=postgres
Password=1

[Paths]
roots=c:\\Tmp\\Work\\Documents,c:\\Tmp\\Work 2\\Documents
extensions="txt,md,ini,csv"
```

Пояснения:

- `Host` — адрес сервера PostgreSQL.
- `Port` — порт PostgreSQL.
- `Name` — имя базы данных.
- `User` / `Password` — логин и пароль пользователя БД.
- `roots` — корневые каталоги для индексации, несколько путей задаются через запятую.
- `extensions` — расширения файлов для индексации (через запятую).
