# Qt Diplom: структура проекта

Этот файл описывает, где что лежит в проекте и для чего это нужно.

## Основные папки

- `Source/DatabaseManager` — консольный проект для работы с БД: подключение, создание таблиц, тестовые операции.
- `Source/indexer` — консольный индексатор: читает файлы из каталогов, считает частоту слов и сохраняет результат в БД, использует DatabaseManager.
- `Source/search` — поисковик на Qt Widgets: принимает запросы, показывает найденные документы и вкладку словаря, использует DatabaseManager.
- `Distributiv` — готовый комплект для запуска (`indexer.exe`, `search.exe`, `config.ini`, Qt DLL, плагины, `libpq.dll`).

## Где что настраивается

- Параметры БД и пути индексации задаются в `config.ini`.
- Для всех приложений важна секция `[Database]` (`Host`, `Port`, `Name`, `User`, `Password`).
- Для индексатора важна секция `[Paths]` (`roots`, `extensions`).


## Порядок работы

1. Заполнить `config.ini` (БД и каталоги).
2. Запустить `indexer` для построения индекса.
3. Запустить `search` для выполнения запросов.
